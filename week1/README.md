# Week 1: Foundations of Data Engineering

## Learning Objectives

By the end of this week, you will be able to:

- Describe what a data engineer does and how it differs from a data analyst and data scientist
- Explain the modern data stack and the data lifecycle
- Have a working local development environment (PostgreSQL + pgAdmin + Git)
- Load a CSV file into a PostgreSQL database
- Write basic SQL queries using `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`, and `DISTINCT`
- Track your coursework in a Git repository connected to GitHub
